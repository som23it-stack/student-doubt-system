# Smart Student Help & Doubt System

This is a full stack web application where students can ask doubts and teachers/admin can reply.

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MongoDB
- Firebase Authentication

## Features
- Student Login/Register
- Ask doubts
- View answers
- Admin panel to reply
- Status: Pending / Answered

## How to Run the Project

1. Install Node.js
2. Install MongoDB and start it
3. Open project folder in terminal
4. Run command:
   npm install
5. Start server:
   node server.js
6. Open index.html in browser

Server runs on:
http://localhost:5000
